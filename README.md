# Dynamic Food Webs

A C-based program that models and simulates dynamic food web ecosystems,
tracking predator-prey relationships and population changes across a 
network of species.

## Overview
Food webs are complex networks that represent feeding relationships 
between organisms in an ecosystem. This project simulates those 
relationships dynamically — modeling how the addition or removal of 
a species ripples through an entire ecosystem.

## Features
- Models predator-prey relationships between multiple species
- Dynamically updates the food web as species are added or removed
- Tracks extinction cascades and population impacts
- Traverses and queries the web using graph-based data structures

## Concepts Demonstrated
- Graph representation and traversal in C
- Dynamic memory allocation and pointer management
- Struct-based data modeling
- Recursive algorithms for network analysis
